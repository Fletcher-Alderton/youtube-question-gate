# Privacy Policy

YouTube Question Gate does not collect, sell, share, transmit, or remotely store personal data.

## Data Stored Locally

The extension stores the following data in browser extension storage on your device:

- Uploaded question sheets.
- Sheet enabled/disabled settings.
- Extension preferences required to run the question gate.

This data stays in your browser profile and is not sent to the developer or any external server by the extension.

## Website Access

The extension runs on YouTube pages so it can show the question gate and pause videos until a question is answered.

The extension does not transmit YouTube browsing activity, video history, answers, uploaded sheets, or settings to any external service.

## Permissions

The extension uses the `storage` permission to save question sheets and settings locally.
